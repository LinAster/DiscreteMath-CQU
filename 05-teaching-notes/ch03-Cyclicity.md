# 循环性 —— 一个"反着转"的性质

> 循环性（circularity）单独看很奇怪，但一旦和自反性搭配，就变成等价关系。2024 年考了它的证明，是真题。

---

## 一、循环性是什么

### 定义

$$ \langle a, b \rangle \in R \ \land \ \langle b, c \rangle \in R \quad \Rightarrow \quad \langle c, a \rangle \in R $$

### 用人话

如果 a → b 且 b → c，那么**c 必须能回到 a**。

不是"a 走到 c"（那是传递性），而是**"绕一圈回到起点"**。

---

## 二、和传递性对比（最容易搞混的地方）

| 性质 | 前提 | 结论 | 方向 |
|------|------|------|:----:|
| **传递性** | a→b, b→c | a→c | 顺着走 |
| **循环性** | a→b, b→c | **c→a** | **反着绕回来** |

画成图：

```
传递性：a ──→ b ──→ c       循环性：a ──→ b ──→ c
         ↘_________↑                 ↑___________↙
           (a→c)                       (c→a)
```

循环性的箭头**从最后一个回到第一个**，形成一个 loop。

---

## 三、一个具体例子

A = {1, 2, 3}，R = {⟨1,2⟩, ⟨2,3⟩, ⟨3,1⟩}

检查循环性：

| a | b | c | a→b? | b→c? | 要求 c→a? | 实际有 c→a? |
|:--:|:--:|:--:|:----:|:----:|:---------:|:----------:|
| 1 | 2 | 3 | ✓ | ✓ | 3→1 | ✓ |
| 2 | 3 | 1 | ✓ | ✓ | 1→2 | ✓ |
| 3 | 1 | 2 | ✓ | ✓ | 2→3 | ✓ |
| 1 | 2 | 1 | ✓ | ✓(2→1 ✗) | — | — |

只有三组同时满足两个前提，结论全部成立。

**R 是循环的** ✓

但注意：R 不是自反的（对角线全 0），所以**单独循环性不能推出等价关系**。

---

## 四、再一个例子：不是循环的

A = {1, 2, 3}，R = {⟨1,2⟩, ⟨2,3⟩, ⟨1,3⟩}

这是传递的（a→b→c 则 a→c），但是：

取 a=1, b=2, c=3：
- 1→2 ✓, 2→3 ✓
- 要求 **3→1**？ ✗

**R 是传递的但不是循环的。**

所以结论：**传递性和循环性互不蕴含。**

---

## 五、核心定理：自反 + 循环 = 等价关系

这是 2024 年 Q18 真题，分值约 8 分。

### 定理陈述

若 R 满足自反性和循环性，则 R 是等价关系。

### 证明（分两步）

**已知**：自反（∀a, ⟨a,a⟩ ∈ R）和循环。

**要证**：对称性 + 传递性。

---

**【第一步：证对称性】**

要证：⟨a,b⟩ ∈ R ⇒ ⟨b,a⟩ ∈ R。

由自反性，⟨b,b⟩ ∈ R。

现在我们有：⟨a,b⟩ ∈ R 且 ⟨b,b⟩ ∈ R。

应用循环性（a 和 b 作为"a→b→c"中的 a,b,c，其中 c=b）：

$$ \langle a, b \rangle \land \langle b, b \rangle \ \Rightarrow \ \langle b, a \rangle $$

✓ 对称性得证。

---

**【第二步：证传递性】**

要证：⟨a,b⟩ ∈ R 且 ⟨b,c⟩ ∈ R ⇒ ⟨a,c⟩ ∈ R。

由循环性：⟨a,b⟩ ∧ ⟨b,c⟩ ⇒ ⟨c,a⟩ ∈ R。

再由**刚证的对称性**：⟨c,a⟩ ∈ R ⇒ ⟨a,c⟩ ∈ R。

✓ 传递性得证。

---

**对称性 + 传递性 + 已知的自反性 → R 是等价关系。** ∎

---

## 六、证明的直觉理解

为什么自反 + 循环能推出这么多东西？

- **自反** = 每个人手里都有"自己这张牌"
- **循环** = 三个人传一圈，能回到起点

把自反这张牌塞进循环里：
- 证明对称性：把 b 自己当"第三个人"塞进去
  - a→b, b→b → 循环要求 b→a
- 证明传递性：先用循环得到 c→a，再用对称性翻过来得到 a→c

**自反提供了"第三人"，循环完成了"绕圈"**。两者一合力，对称和传递就都出来了。

## 七、证明的"模板文字"（考试直接套）

题干：若 R 是自反的且循环的，证明 R 是等价关系。

```
证：
（1）对称性
  设 ⟨a,b⟩ ∈ R。
  由自反性，⟨b,b⟩ ∈ R。
  对 ⟨a,b⟩ 和 ⟨b,b⟩ 应用循环性，得 ⟨b,a⟩ ∈ R。
  故对称性成立。

（2）传递性
  设 ⟨a,b⟩ ∈ R 且 ⟨b,c⟩ ∈ R。
  由循环性，得 ⟨c,a⟩ ∈ R。
  由（1）已证的对称性，⟨c,a⟩ ∈ R ⇒ ⟨a,c⟩ ∈ R。
  故传递性成立。

综上，R 满足自反性、对称性、传递性 ⟹ R 是等价关系。  ∎
```

## 八、常见混淆（避坑）

| 易混 | 澄清 |
|------|------|
| 循环性能不能推出传递性？ | **不能**。必须要有自反性搭桥。纯循环 R = {⟨1,2⟩, ⟨2,3⟩, ⟨3,1⟩} 不是传递的：1→2, 2→3 但 1→3 不在。 |
| 传递性能不能推出循环性？ | **不能**。R = {⟨1,2⟩, ⟨2,3⟩, ⟨1,3⟩} 传递但不循环。 |
| 有没有既循环又传递但不等价的？ | 有。R = ∅ 在 A={1,2} 上：循环 ✓（前提永假），传递 ✓，但不自反 → 不等价。 |
| 循环性考的多吗？ | 不多，但 2024 年出了，说明这是一个**新考点**，2026 可能再来。 |
