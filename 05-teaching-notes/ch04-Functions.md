# 第4章 函数

> 函数是最短的一章（2次作业，考试占约3%），但概念必须清晰——单射/满射/双射的理解直接关系到后续代数结构的学习。

---

## 4.1 函数是什么

**函数就是特殊的关系**：定义域中每个元素，恰好对应值域中的一个元素。

$$f: A \to B$$

A = **定义域**（domain），B = **陪域**（codomain）。

**函数的两个要求**（缺一不可）：
1. **处处定义**：A 中每个元素都有像。A 中不能有"落单"的。
2. **唯一性**：A 中每个元素只有一个像。不能"脚踏两只船"。

**三个关键集合**：

| 概念 | 符号 | 含义 |
|------|:----:|------|
| **定义域** | $\text{dom}(f)$ | $f$ 可以"吃"的所有输入 |
| **陪域** | $\text{codom}(f)$ | $f$ 输出的"理论上范围"（规定了 B） |
| **值域** | $\text{ran}(f)$ | $f$ 输出的"实际范围" = $\{f(x) \mid x \in A\}$ |

> [!tip] 关键区别
> 值域 $\subseteq$ 陪域。值域是"真的用到了的那些输出"，陪域是"声明的输出范围"。
>
> 就像"声称你能跑100米以内"（陪域），但"你最多只跑过50米"（值域）。

## 4.2 三种函数类型

**这是必考概念。用"输入→输出"的直观理解来记。**

### 4.2.1 单射（injective，一一到）

**不同的输入 → 不同的输出。**

$$f(a) = f(b) \Rightarrow a = b$$

**通俗说法**：不会有两个不同的输入落到同一个输出上。

**图示理解**：

```
单射 ✓：               非单射 ✗：
A       B              A       B
1 ---→ a              1 ---→ a
2 ---→ b              2 ---→ b
3 ---→ c              3 ---╯
                        （1和3都到a）
```

### 4.2.2 满射（surjective，映上的）

**陪域中每个元素都被打到。**

$$\forall y \in B, \exists x \in A, f(x) = y$$

**通俗说法**：输出范围 = 陪域。声称能达到的都达到了。

```
满射 ✓：               非满射 ✗：
A       B              A       B
1 ---→ a              1 ---→ a
2 ---→ b              2 ---→ b
3 ---→ c              3 ---→ c
                        （d没被任何x打到）
```

### 4.2.3 双射（bijective）

**既是单射，又是满射 → 一一对应。**

```
双射 ✓：
A       B
1 ←--→ a
2 ←--→ b
3 ←--→ c
```

每一个 B 中的元素恰好被一个 A 中元素打到。不多不少，完美配对。

### 4.2.4 判定速查表

| 性质 | 口诀 | 对等式的判定条件 |
|------|------|------------------|
| **单射** | 不同进，不同出 | $f(a) = f(b)$ 必有 $a = b$ |
| **满射** | 全都打到 | $\forall y \in B, f^{-1}(y) \neq \emptyset$ |
| **双射** | 一一对应 | 既单又满 |

### 4.2.5 有限集上的计数关系

若 $|A| = m$，$|B| = n$：

- 存在单射 → **必须** $m \leq n$（A 的元素不能比 B 多）
- 存在满射 → **必须** $m \geq n$（A 的元素不能比 B 少）
- 存在双射 → **必须** $m = n$（一样多）

## 4.3 复合函数

$$(g \circ f)(x) = g(f(x))$$

先做 $f$，再做 $g$。注意顺序：$(g \circ f)$ 是从右往左读，先 $f$ 后 $g$。

### 4.3.1 复合函数的性质

这是最常考的部分。已知 $f: A \to B$，$g: B \to C$，判断 $g \circ f$ 的性质。

**记住这张表（T=成立，F=不一定）**：

| $f$ 的性质 | $g$ 的性质 | $g \circ f$ 是单射？ | $g \circ f$ 是满射？ |
|:----------:|:----------:|:-------------------:|:-------------------:|
| 单射 | 单射 | **T** | F |
| 满射 | 满射 | F | **T** |
| 双射 | 双射 | **T** | **T** |

**口诀**：两个单复合还是单，两个满复合还是满。

### 4.3.2 逆推性质

| 条件 | 结论 |
|------|------|
| $g \circ f$ 是单射 | **$f$ 一定是单射**（$g$ 不一定） |
| $g \circ f$ 是满射 | **$g$ 一定是满射**（$f$ 不一定） |
| $g \circ f$ 不是满射 | $g$ 肯定不是满射 |

**为什么 $g \circ f$ 单射 → $f$ 单射？**

反证：如果 $f$ 不是单射，存在 $a \neq b$ 但 $f(a) = f(b)$。
则 $(g \circ f)(a) = g(f(a)) = g(f(b)) = (g \circ f)(b)$，
但 $a \neq b$ → $g \circ f$ 不是单射，矛盾。

**真题（2024 Q6）**：$f: X \to Y$，$g: Y \to Z$

```
正确选项：
A. f 和 g 都是满射 ⇒ g∘f 是满射  ✓
B. g∘f 不是满射 ⇒ g 不是满射       ✓（逆否：若g是满射则...不一定）

错误选项：
C. f 是满射 ⇒ g∘f 是满射  ✗（g 可能不是满射）
D. g 是满射 ⇒ g∘f 是满射  ✗（f 的值域可能不是整个Y）
```

## 4.4 逆函数

**只有双射才有逆函数。**

$$f^{-1}: B \to A, \quad f^{-1}(y) = x \text{ 当 } f(x) = y$$

**逆函数的两条性质**：
- $f^{-1} \circ f = I_A$（恒等函数 on A）
- $f \circ f^{-1} = I_B$（恒等函数 on B）

**注意**：不要把逆函数 $f^{-1}$ 和倒数 $\frac{1}{f}$ 混淆。在离散数学里 $f^{-1}$ 不是"1除以f"，而是"反过来映射"。

## 4.5 本章知识地图

```
函数
├── 基本概念
│   ├── 定义域、陪域、值域（值域⊆陪域）
│   └── 函数的两个要求（处处定义+唯一性）
│
├── 三种类型
│   ├── 单射：不同输入→不同输出
│   ├── 满射：陪域全被打到
│   └── 双射：既单又满（才有逆函数）
│
├── 复合函数
│   ├── (g∘f)(x) = g(f(x))
│   ├── 单+单→单，满+满→满
│   └── g∘f单→f必单，g∘f满→g必满
│
└── 逆函数
    ├── 仅双射有逆
    └── f⁻¹∘f = I_A, f∘f⁻¹ = I_B
```
