# 陪集 —— aH 之间到底是什么运算

> 陪集卡住几乎都是同一个原因：群运算的符号太抽象。本文用**具体数字**拆到底。

---

## 一、aH 不是"两个东西在运算"

先纠正一个最常见的误解：

- $a$ 是群里的**一个元素**（比如数字 3）
- $H$ 是一个**子群**，里面有一堆元素（比如 {0, 2, 4, 6}）
- $aH$ 是把 $a$ 和 $H$ 里的**每一个元素**都用群运算算一遍，得到**一个新的集合**

**aH 是一个集合，不是一个运算结果。**

---

## 二、群里的运算只有一个

群 $(G, *)$ 里面，**只有一个运算**，通常记作 $*$。

根据上下文，它可能是：
- 加法 +（整数群、模 n 加法群）→ 读作"a 加每个"
- 乘法 ×（非零实数乘法群、模 p 乘法群）→ 读作"a 乘每个"
- 矩阵乘法、函数复合……

**aH 里的运算就是群自己的那个运算。不需要新运算，也没有新运算。**

写成：$aH = \{ a * h \mid h \in H \}$

---

## 三、加法群例子（最直观）

群 G = $\mathbb{Z}_{8}$ = {0, 1, 2, 3, 4, 5, 6, 7}，运算 = **模 8 加法**。

子群 H = {0, 4}（0+0=0, 4+4=0 mod 8，确实封闭）。

### 取 a = 0：

$$ 0H = \{0+0, \ 0+4\} = \{0, 4\} = H $$

> 单位元的陪集就是子群本身。

### 取 a = 1：

$$ 1H = \{1+0, \ 1+4\} = \{1, 5\} $$

把 H = {0, 4} 整体"平移"了两个位置（在 mod 8 的世界里）。

### 取 a = 2：

$$ 2H = \{2+0, \ 2+4\} = \{2, 6\} $$

### 取 a = 3：

$$ 3H = \{3+0, \ 3+4\} = \{3, 7\} $$

### 四个陪集一览

```
H   = 0H = {0, 4}
1H  =     {1, 5}
2H  =     {2, 6}
3H  =     {3, 7}
```

- 四个陪集**大小完全相同**（都是 2 个元素）
- 它们**互不相交**
- 全部拼起来刚好是 G = {0,1,2,3,4,5,6,7}

这就是陪集背后的几何直觉：**用子群把整个群切成等大的片**。

---

## 四、乘法群例子

群 G = $\mathbb{Z}_7^*$ = {1, 2, 3, 4, 5, 6}，运算 = **模 7 乘法**。

子群 H = {1, 6}（因为 6×6 = 36 ≡ 1 mod 7）。

### 取 a = 1：

$$ 1H = \{1 \times 1, \ 1 \times 6\} = \{1, 6\} = H $$

### 取 a = 2：

$$ 2H = \{2 \times 1, \ 2 \times 6\} = \{2, \ 12 \bmod 7\} = \{2, 5\} $$

$12 \div 7 = 1 \dots 5$ → 12 mod 7 = 5。

### 取 a = 3：

$$ 3H = \{3 \times 1, \ 3 \times 6\} = \{3, 18 \bmod 7\} = \{3, 4\} $$

### 三个陪集

```
H  = 1H = {1, 6}
2H =     {2, 5}
3H =     {3, 4}
```

同样是切成等大的片，每片 2 个元素。

## 五、左陪集 vs 右陪集

| 类型 | 写法 | 谁在左边 | 运算 |
|:----:|------|----------|------|
| 左陪集 | $aH$ | a 在左 | a * h |
| 右陪集 | $Ha$ | a 在右 | h * a |

**如果群运算是交换的**（比如加法群），左陪集 = 右陪集，因为 a+h = h+a。不用区分。

**如果群运算不交换**（比如矩阵乘法），左陪集和右陪集可能不同。但离散数学考试中，基本都是交换群，区分不区分都一样。

## 六、陪集的关键性质（考试重点）

| 性质 | 含义 |
|------|------|
| $|aH| = |H|$ | 每个陪集大小和子群一样 |
| $aH = bH$ 或 $aH \cap bH = \emptyset$ | 两个陪集要么完全相同，要么不交 |
| 所有左陪集的并 = G | 陪集把 G 完整划分 |
| $aH = H \iff a \in H$ | a 在子群里 → 它的陪集就是子群本身 |

---

## 七、拉格朗日定理（为什么学陪集）

$$ |G| = |H| \times [G:H] $$

- $|G|$：群的元素个数（阶）
- $|H|$：子群的元素个数
- $[G:H]$：**不同陪集的个数**（指数）

上面模 8 加法例子里：8 = 2 × 4。子群大小 2，4 个陪集。

**推论**：子群的阶必须整除群的阶。一个 15 阶的群里不可能有 4 阶子群（15 ÷ 4 不是整数）。

---

## 八、从集合角度看陪集（加深理解）

如果把群想象成一张地图，子群就是一个模板：

```
G = { 0, 1, 2, 3, 4, 5, 6, 7 }   ← 整张地图
H = { 0, 4 }                       ← 模板

把模板"平移"到每个位置：
  a=0: {0, 4}
  a=1: {1, 5}
  a=2: {2, 6}
  a=3: {3, 7}
```

每个陪集就是模板的一个"平移副本"。模板平移到任何位置，形状不变。

---

## 九、常见坑

| 坑 | 澄清 |
|------|------|
| 把 aH 当成一次运算 | aH 是一个**集合**，是把 H 里每个元素都和 a 算一遍的**结果集合** |
| 以为需要新运算 | 不需要。就用群本身的运算，加群用加法，乘群用乘法 |
| $aH = Ha$ 总是成立 | 只在**交换群**中一定成立。非交换群需要小心 |
| 忘了取模 | 有限群（如 ℤₙ）运算后要**取模**，3+5 mod 6 = 2，不是 8 |
