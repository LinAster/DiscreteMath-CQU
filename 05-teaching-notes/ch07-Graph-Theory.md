# 第7章 图论

> 图论研究"点和线组成的网络"。地图、社交网络、电路板布线——背后都是图论。
> 这一章重量级（7次作业，考试占约25%），但概念比代数结构更直觉。

---

## 7.1 图的基本概念

### 7.1.1 什么是图

图由两部分组成：**顶点（点）** 和 **边（连线）**。

$$G = (V, E)$$

- $V$ = 顶点集合（Vertices）
- $E$ = 边集合（Edges），每条边连接两个顶点

```
简单图 G：
    1 --- 2
    |     |
    3 --- 4
```

V = {1, 2, 3, 4}，E = {{1,2}, {1,3}, {2,4}, {3,4}}

### 7.1.2 图的分类

| 类型 | 边的形式 | 通俗说法 |
|------|----------|----------|
| **无向图** | 边没有方向 | 双向车道 |
| **有向图** | 边有方向（箭头） | 单行道 |
| **简单图** | 无自环、无重边 | "规矩的"图 |
| **多重图** | 允许多条边连接同一对顶点 | "网状"的图 |
| **完全图 $K_n$** | 每对顶点之间都有边 | "全通"图，$n$ 个点有 $\frac{n(n-1)}{2}$ 条边 |

### 7.1.3 握手定理（最基础但最重要的公式）

$$\sum_{v \in V} \deg(v) = 2|E|$$

**所有顶点的度数之和 = 边数的两倍。**

为什么？因为每条边连接两个顶点，每连一次就贡献两度。

**通俗理解**：一个聚会上所有人握手的次数加起来，等于握手次数的两倍（每次握手涉及两个人）。

**推论**：奇度顶点的个数一定是**偶数**。

> 证明：度数总和 = $2|E|$ 是偶数。把顶点按奇偶分成两组：偶度之和是偶数，奇度之和 = 总偶 - 偶度之和 = 偶数。而奇数个奇数之和是奇数，所以奇度顶点必须是偶数个。

**真题（2023 Q12）**：结点数为奇数，且所有结点度数都为奇数的图——

```
奇数个奇数之和 = 奇数 ≠ 2|E|（偶数）
→ 这样的图不存在！
→ 答案：D（不存在）
```

**真题（2024 Q10）**：4顶点简单连通图，3个顶点度数为 2, 3, 3。第4顶点度数？

```
2 + 3 + 3 + d₄ = 2|E| = 偶数
8 + d₄ 是偶数 → d₄ 是偶数

简单图 → d₄ ≤ 3（最多连3条）
连通图 → d₄ ≥ 1

∴ d₄ = 2（唯一解）
```

## 7.2 路径、回路与连通性

### 7.2.1 基本概念

| 概念 | 含义 | 记忆方法 |
|------|------|----------|
| **路径** | 一连串首尾相连的边 | 不重复走顶点 |
| **简单路径** | 路径中顶点不重复 | "不走回头路" |
| **回路** | 起点和终点相同的路径 | "回到家了" |
| **欧拉回路** | 经过每条**边**恰好一次 | 一笔画 |
| **哈密尔顿回路** | 经过每个**顶点**恰好一次 | 旅行商 |

### 7.2.2 连通性

- **连通图**：任意两顶点之间都有路径
- **连通分量**：极大连通子图（"一片大陆"）
- **割点/桥**：删掉它图就不连通了

## 7.3 有向图与矩阵表示

### 7.3.1 邻接矩阵

有向图 $G = (V, E)$，$|V| = n$，邻接矩阵 $A$ 是 $n \times n$ 矩阵：

$$A[i][j] = \begin{cases} 1 & \text{存在从 } v_i \text{ 到 } v_j \text{ 的边} \\ 0 & \text{否则} \end{cases}$$

### 7.3.2 邻接矩阵的幂——强大的工具

**$A^k[i][j]$ = 从 $v_i$ 到 $v_j$ 长度为 $k$ 的路径条数。**

```
A     = 长度为 1 的路径数（就是直接的边）
A²    = 长度为 2 的路径数
A³    = 长度为 3 的路径数
```

- **长度为 $k$ 的所有路径总数** = $A^k$ 的所有元素之和
- **长度为 $k$ 的所有回路个数** = $A^k$ 的对角线元素之和

### 7.3.3 可达性矩阵

$$P = (A \lor A^2 \lor A^3 \lor \dots \lor A^n)^{\text{布尔}}$$

用布尔运算（1+1=1），$P[i][j] = 1$ 表示从 $v_i$ 可以到达 $v_j$（路径长度不限）。

## 7.4 欧拉图与哈密尔顿图

### 7.4.1 欧拉图

> 欧拉回路：经过**每条边**恰好一次，最后回到起点。

**判定定理（记住就行）**：

| 图的类型 | 欧拉回路存在的条件 | 欧拉路径存在的条件 |
|----------|---------------------|---------------------|
| **无向图** | 所有顶点度数为**偶数** | 恰好 0 或 2 个奇度顶点 |
| **有向图** | 每个顶点入度 = 出度 | 至多一顶点入=出+1，至多一顶点出=入+1 |

### 7.4.2 哈密尔顿图

> 哈密尔顿回路：经过**每个顶点**恰好一次。

**注意**：哈密尔顿图的判定没有简单的充要条件（NP-完全问题）。考试中通常**给图判断**或用必要条件排除。

**必要条件（用度数排除）**：
- 若图中有度数小于 2 的顶点，则不可能是哈密尔顿图
- 哈密尔顿图中删去 $k$ 个顶点后，连通分量数 $\leq k$

## 7.5 树与最小生成树

### 7.5.1 树

**树是无回路的连通图。**

树的等价定义（以下都是等价的）：
1. 连通且无回路
2. 连通且有 $n-1$ 条边
3. 无回路且有 $n-1$ 条边
4. 任意两顶点间有唯一简单路径

**核心公式**：$n$ 个顶点的树恰好有 $n-1$ 条边。

### 7.5.2 生成树

**生成树**：包含图中所有顶点的树（连通无回路子图）。

每个连通图至少有一棵生成树。生成树 = 原图去掉一些边，剩下刚好 $n-1$ 条且连通。

### 7.5.3 最小生成树（MST）—— Kruskal 算法

**问题**：每条边有"代价"（权值），找一棵生成树让总代价最小。

**Kruskal 算法（口诀：从小到大不吃回头路）**：

```
Step 1: 把所有边按权值从小到大排序

Step 2: 从最小的开始，依次考虑每条边：
        如果加入这条边不会形成回路 → 采用
        如果会形成回路 → 跳过

Step 3: 重复，直到采用了 n-1 条边
```

**真题（2024 Q16）**：8 顶点有权图，求 MST 和树权。

直接按 Kruskal 做——边权排序为 1, 1, 1, 1, 2, 2, 2, ...

```
选边顺序：
L-ML(2): 权2 ✓
TL-ML(1): 权1 ✓
BL-BR(1): 权1 ✓
BR-R(1): 权1 ✓
TR-MR(1): 权1 ✓
MR-BR(2): 权2 ✓
ML-BR(2): 权2 ✓

共 7 条边（8-1=7），树权 = 2+1+1+1+1+2+2 = 10
```

**考试要求**：必须把每一步选取的子图**画出来**，不画会扣分。

### 7.5.4 Prim 算法（另一种方法）

```
Step 1: 任选一个起始顶点
Step 2: 在所有"一端已选、一端未选"的边中，选权值最小的
Step 3: 把新顶点加入"已选集"
Step 4: 重复直到所有顶点都入选
```

两种算法结果一样（前提是边权互异），树权相同。

## 7.6 平面图与欧拉公式

### 7.6.1 什么是平面图

**平面图**：能画在平面上且边与边之间不相交的图。

比如 $K_4$（4 个顶点的完全图）是平面图：

```
 1 --- 2
 | \ / |
 |  X  |
 | / \ |
 4 --- 3

画法改进（不相交）：
   1
  /|\
 / | \
4--3--2
```

### 7.6.2 欧拉公式（平面图第一定理）

$$v - e + r = 2$$

- $v$ = 顶点数
- $e$ = 边数
- $r$ = 面数（包括外面那个无限大面）

### 7.6.3 平面图的边数上限

| 条件 | 上限不等式 |
|------|-----------|
| 简单连通平面图 | $e \leq 3v - 6$ |
| 无三角形（面至少 4 条边） | $e \leq 2v - 4$ |

**推导思路**：每个面至少 3 条边 → $\sum \deg(f) \geq 3r$，而 $\sum \deg(f) = 2e$（每条边被两面对称），所以 $2e \geq 3r$。代入 $r = e - v + 2$，得 $e \leq 3v - 6$。

### 7.6.4 平面图证明套路

这是考试证明题的热点。**标准套路**：

```
已知条件 → 不等式约束 → 代入欧拉公式 → 导出矛盾或结论
```

**真题（2024 Q20）**：$G$ 是简单连通平面图，$r < 12$，每个顶点度 $\geq 3$。证明存在面次数 $\leq 4$。

```
证：

设 n = 顶点数, m = 边数, r = 面数

(1) 欧拉公式：n - m + r = 2

(2) 最小度 ≥ 3：
    握手定理：∑deg(v) = 2m
    每个顶点度 ≥ 3 → 3n ≤ 2m → n ≤ 2m/3

(3) r < 12

    将(2)代入(1)：2 < 2m/3 - m + 12
    → m < 30

(4) 反证法：假设所有面次数 ≥ 5
    则 5r ≤ ∑deg(f) = 2m → r ≤ 2m/5

    将(2)和(4)代入(1)：
    2 ≤ 2m/3 - m + 2m/5 = (10m - 15m + 6m)/15 = m/15
    → m ≥ 30

(5) 矛盾：m < 30 且 m ≥ 30 同时成立。

所以假设不成立，必存在面次数 ≤ 4。 ∎
```

**评分点**：写欧拉公式得1分，度约束得2分，面次数假设得2分，代入导出矛盾得2分。

## 7.7 图同构

### 7.7.1 定义

两个图**同构**，就是"外表不同但骨架一样"——可以通过重命名顶点使两个图完全一致。

### 7.7.2 判断方法

**快速排除法（必要条件）**：

```
1. 顶点数不同 → 不同构
2. 边数不同 → 不同构
3. 度数序列不同（排序后比较） → 不同构
4. 子图结构不同 → 不同构
```

如果以上全部吻合，再尝试建立顶点的一一映射。

**真题（2024 Q11）**：选出与给定图同构的图。排除法先过前三关，剩余手工验证。
