# 全序与良序 —— 从偏序往上升级

> 全序和良序都是**特殊的偏序**。理解它们的唯一方法：先懂偏序，再看多出来的那条要求。

---

## 零、偏序（复习十秒）

偏序 = 自反 + 反对称 + 传递。

记作 $\preccurlyeq$，读作"小于等于"。

**偏序为什么叫"偏"？**——因为不一定每两个元素都能比大小。

---

## 一、"可比"到底是什么意思

### 能比：$\{1\}$ 和 $\{1,2\}$

集合 A 的幂集上，定义偏序为"包含于 ⊆"。

$\{1\} \subseteq \{1,2\}$ ✓ —— $\{1\}$ 比 $\{1,2\}$ 小（被包含），能比。

### 不能比：$\{1\}$ 和 $\{2\}$

$\{1\} \not\subseteq \{2\}$，$\{2\} \not\subseteq \{1\}$ —— 谁也不包含谁，**不能比**。

这就是"偏"的意思：**有一些对能比，有一些不能比**。

---

## 二、全序（线序）—— 所有人都能比

### 定义

全序 = 偏序 + **每两个元素都可比较**。

即可比性条件：$\forall a, b \in A$，要么 $a \preccurlyeq b$，要么 $b \preccurlyeq a$。

### 全序的例子 —— 数字的大小

A = {1, 3, 5, 7}，偏序 = 通常的 ≤。

- 1 和 3 能比 ✓（1 ≤ 3）
- 3 和 7 能比 ✓
- **任意两个都能比** ✓

所以 (A, ≤) 是一个**全序**。

### 不是全序的例子 —— 包含关系（同一个例子再看一遍）

A = { {1}, {2}, {1,2} }，偏序 = ⊆。

- {1} 和 {2} 能比吗？不能。谁也不包含谁。

所以 (A, ⊆) 是**偏序但不是全序**。

### 为什么又叫线序

因为全序的元素可以**排成一条线**：

```
1 —— 3 —— 5 —— 7
```

每个位置上的元素和所有其他元素的相对大小都确定了。

---

## 三、全序 vs 偏序 —— 判定方法

给你一个关系求判定，三步走：

| 步骤 | 检查什么 | 通过 |
|------|----------|:--:|
| 1 | 自反 + 反对称 + 传递？ | 是偏序 |
| 2 | 任意两元素都能比？ | 是全序 |
| 3 | 每个非空子集都有最小元？ | 是良序 |

**卡在步骤 2 → 偏序但不是全序。通过步骤 2 但卡在步骤 3 → 全序但不是良序。**

---

## 四、良序 —— 全序的严格升级

### 定义

良序 = 全序 + **每个非空子集都有最小元**。

### 例子 1：自然数集（良序）

A = $\mathbb{N} = \{0, 1, 2, 3, \dots\}$，偏序 = ≤。

随便取一个非空子集：
- 子集 B = {5, 9, 100, 3} → 最小元 = 3 ✓
- 子集 B = {0, 1, 2, 3, ...}（无限集）→ 最小元 = 0 ✓
- **任何非空子集**，因为自然数有下界 0，总能找到最小那个 ✓

所以 ($\mathbb{N}$, ≤) 是**良序**。

### 例子 2：整数集（全序但不是良序）

A = $\mathbb{Z} = \{\dots, -2, -1, 0, 1, 2, \dots\}$，偏序 = ≤。

- 任意两个整数能比吗？能。所以它是**全序**。

但取子集 B = {所有负整数} = {...，-3, -2, -1}：
- 有最小元吗？**没有。** 因为负整数无限往下延伸（-1 > -2 > -3 > ... 一直小下去）。

所以 ($\mathbb{Z}$, ≤) 是**全序但不是良序**。

### 例子 3：正实数区间（全序但不是良序）

A = (0, 1) = {x | 0 < x < 1}，偏序 = ≤。

- 任意两个能比 ✓（全序）
- 取子集 B = A 本身：有最小元吗？**没有。** 0 不在 A 里，而 A 中任何一个数都可以再取一半得到更小的数。

所以 ((0, 1), ≤) 是**全序但不是良序**。

---

## 五、全序 vs 良序 —— 反例对照

| 集合 | 偏序 | 全序？ | 良序？ | 为什么卡住 |
|------|:----:|:-----:|:-----:|-----------|
| {1, 3, 5, 7} | ≤ | ✓ | ✓ | 有限全序 = 自动良序 |
| $\mathbb{N}$ | ≤ | ✓ | ✓ | 每个子集有最小元（下界为 0） |
| $\mathbb{Z}$ | ≤ | ✓ | ✗ | 负整数子集无最小元 |
| (0, 1) 开区间 | ≤ | ✓ | ✗ | 无限逼近 0 但 0 不在集合里 |
| $\mathcal{P}(\{1,2\})$ | ⊆ | ✗ | ✗ | 第一步就卡：{1} 和 {2} 不能比 |

## 六、一个重要的特殊情况

**有限全集上的全序一定是良序。**

因为有限个元素，任何非空子集也是有限的，逐个比较总能找到最小的。

所以良序的"难点"全在**无限集**上——有限集上全序和良序是等价的。

## 七、考题中的出现形式

离散数学考试中，全序和良序通常出现在：

1. **判断题**：给你一个集合和一个偏序，问是不是全序、是不是良序
2. **哈斯图题**：给你哈斯图，看图判断
3. **证明题**：证某个结构是良序（较少见）

### 哈斯图快速判断法

哈斯图是全序的充要条件：所有元素在**同一条链**上（图中没有分叉）。

```
全序（排成一列）：       不是全序（有分叉）：
  5                      5
  │                     / \
  4                    3   4
  │                    |   |
  3                    1   2
  │
  2
  │
  1
```

左边每个元素都能和任何其他元素比（通过上下关系），右边 3 和 4 不能比。
