# 对偶图 & 图的正常着色

---

## 第一部分：对偶图

### 一、什么是平面图的对偶图

> 给定一个平面图 G，把每个**面**变成一个顶点，两个面之间如果**共享一条边**，就在对偶图中连一条边。得到的新图叫对偶图 G*。

### 二、构造规则

| 步骤 | 操作 |
|---|---|
| 1 | G 的每个面（包括无限面）→ G* 的一个顶点 |
| 2 | G 中每条边 e 隔着两个面 f1, f2 → G* 中连 f1—f2 |
| 3 | 如果一条边两边是同一个面（桥），G* 中就是自环 |

### 三、例子

```
平面图 G：
  a —— b
  |    |
  d —— c

面：f1（内部正方形），f0（外面无限面）
```

对偶图 G*：

```
  f1 —— f0
  (一条边对应原来一条边，4 条边 → G* 有 4 条平行边)
```

实际上 G* 有 2 个顶点、4 条边（每条原图的边跨着 f0 和 f1）。

### 四、更复杂的例子

```
平面图 G（三角形）：
    a
   /\
  /  \
 b—— c

面：f1（内部三角形），f0（外面）
```

G*：2 个顶点、3 条边（f1 和 f0 之间连 3 条平行边）。

### 五、对偶图的关键性质

| 性质 | 含义 |
|---|---|
| G* 的顶点数 = G 的面数 | r* = n, n* = r |
| G* 的边数 = G 的边数 | m* = m |
| (G*)* = G | 对偶再对偶回到原图 |
| **G 中的回路 ⇔ G* 中的割集** | 这是最核心的对偶关系 |
| G 中的割集 ⇔ G* 中的回路 | 反过来也成立 |
| G 连通 → G* 也连通 | |

### 六、核心对偶关系详解

```
G 中的回路   ←→   G* 中的割集
G 中的割集   ←→   G* 中的回路
```

> 这在电路分析中极其重要：平面电路的**回路电流法**和**节点电压法**本质上互为对偶！

## 第二部分：图的正常着色

### 一、什么是正常着色

> 给每个顶点涂一种颜色，使得**任意相邻的两个顶点颜色不同**。

### 二、基本概念

| 概念 | 含义 |
|---|---|
| k-着色 | 用 k 种颜色的正常着色 |
| 色数 χ(G) | 正常着色所需的最少颜色数 |
| k-可着色 | χ(G) ≤ k，即 k 种颜色够用 |

### 三、例子

```
  a —— b
  |    |
  d —— c
```

正方形：a 和 c 对角 → 可以同色，b 和 d 对角 → 可以同色。

最少 2 种颜色：a,c 涂红色，b,d 涂蓝色。χ(G) = 2。

```
完全图 K₃（三角形）：
    a
   /\
  /  \
 b—— c
```

任意两个都相邻 → 必须 3 种不同颜色。χ(K₃) = 3。

### 四、常见图的色数

| 图类型 | 色数 χ(G) | 原因 |
|---|---|---|
| 完全图 Kₙ | n | 所有顶点两两相邻 |
| 二分图 | 2 | 两部分的顶点不相邻 |
| 树 | 2 | 树是二分图 |
| 偶圈 C₂ₖ | 2 | 交替着色 |
| 奇圈 C₂ₖ₊₁ | 3 | 最后一边两头同色 → 需第三色 |
| 轮图 Wₙ (n≥4) | 如果 n 奇 = 4；n 偶 = 3 | 中心点 + 外圈交替 |

### 五、四色定理（著名结论）

> 任何平面图都可以用**最多 4 种颜色**正常着色。即对任何平面图 G，χ(G) ≤ 4。

### 六、色多项式 P(G, k)

用 k 种颜色的着色方案数，是一个关于 k 的多项式。

**例子：K₃（三角形）**

$$P(K_3, k) = k(k-1)(k-2)$$

- 顶点 1：k 种选法
- 顶点 2：k−1 种（不能和 1 同色）
- 顶点 3：k−2 种（不能和 1 或 2 同色）

**例子：P₃（3 个顶点的路）**

$$P(P_3, k) = k(k-1)^2$$

- 顶点 1：k 种
- 顶点 2：k−1 种（和 1 不同）
- 顶点 3：k−1 种（只需和 2 不同，可以和 1 同色）

### 七、色数的基本界

| 界 | 公式 |
|---|---|
| 下界（团数） | χ(G) ≥ ω(G)（最大完全子图的阶） |
| 上界（最大度） | χ(G) ≤ Δ(G) + 1（Brooks 定理） |
| Brooks 特例 | 完全图和奇圈 = Δ+1，其他 ≤ Δ |

## 两张图的对比总结

| | 对偶图 G* | 正常着色 |
|---|---|---|
| 研究对象 | 平面图的面 → 顶点 | 图的顶点 → 颜色 |
| 核心公式 | n* = r, m* = m, r* = n | χ(G) = 最少颜色数 |
| 核心定理 | 回路 ⇔ 割集 | χ(G) ≤ 4（平面图） |
| 应用 | 电路分析（节点⇔回路对偶） | 排课表、寄存器分配、地图着色 |

---

**一句话：对偶图 = 把面变顶点、跨边就连线、回路和割集互换。正常着色 = 相邻顶点不同色，平面图最多 4 色。**
