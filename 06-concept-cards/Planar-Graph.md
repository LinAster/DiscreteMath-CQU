# 平面图及其判定定理

---

## 一、什么是平面图

> 能把图画在平面上，且**边与边只在顶点处相交**（没有边交叉），这个图就是平面图。

### 例子

```
  平面图：              非平面图：
  a —— b               a —— b
  | \  |               | \/ |
  |  \ |               | /\ |
  d —— c               d —— c
                       边 ac 和 bd 不可避免交叉！
```

> 关键：**"能画出来"就行**，不是说现在的画法不交叉，而是存在一种画法不交叉。

---

## 二、欧拉公式（核心公式）

**对连通平面图：**

$$n - m + r = 2$$

| 符号 | 含义 |
|---|---|
| n | 顶点数 |
| m | 边数 |
| r | 面数（包括最外面的无限面） |

### 例子：正方形

```
  a —— b
  |    |
  d —— c
```

n=4, m=4, r=2（内部面 + 外面）→ 4−4+2=2 ✓

### 加上对角线

```
  a —— b
  | \  |
  |  \ |
  d —— c
```

n=4, m=5, 面：两个三角形面 + 外面 = 3 → 4−5+3=2 ✓

---

## 三、平面图的必要条件（用于否决）

由欧拉公式推导出来的两条不等式：

### 3.1 边数限制

**对 n≥3 的连通简单平面图：**

$$m \leq 3n - 6$$

> 边最多是顶点数的 3 倍再减 6。超过 → 一定不是平面图。

### 3.2 无三角形时的强化版

**如果图中没有长度为 3 的回路（即没有三角形）：**

$$m \leq 2n - 4$$

> 二分图（如 K₃,₃）适用这条。

### 用边数否决例子

| 图 | n | m | 3n−6 | 结论 |
|---|---|---|---|---|
| K₅ | 5 | 10 | 9 | 10 > 9 → **不是平面图** ✓ |
| K₃,₃ | 6 | 9 | 12 | 9 ≤ 12，这条没拦住！ |

> K₃,₃ 通过了边数检查，但它确实不是平面图——需要更强的定理。

---

## 四、Kuratowski 定理（充要条件！）

> 一个图是平面图 **⇔** 它不包含 K₅ 或 K₃,₃ 的**细分**（同胚子图）。

### 什么是细分（同胚）？

在边上插入度为 2 的新顶点，把一条边变成一条路：

```
原图边 a——b
细分后 a——x——b（在中间插入 x）
```

> 细分不改变平面性——一条路可以随便弯，不会造成交叉。

### K₅ 和 K₃,₃ 是两个"最小非平面图"

```
   K₅（完全图 5）        K₃,₃（完全二分图）
    1                    a   b   c
   /|\                  |＼ /| ＼ /|
  / | \                  ＼| /＼| /
 2--3--4                 d   e   f
  \ | /
   \|/
    5
```

**这两个图是"非平面图的原子"**——任何一个非平面图，往里缩小边（删除度为 2 的顶点），最终都会缩成 K₅ 或 K₃,₃。

---

## 五、判定流程

```
1. 先用边数检查：m > 3n-6？ → 是 → 不是平面图，结束
2. 如果没有三角形：m > 2n-4？ → 是 → 不是平面图，结束
3. 尝试找 K₅ 或 K₃,₃ 的子图（合并度为 2 的顶点找同胚）
4. 找到了 → 不是平面图
5. 都找不到 → 试着画出来 → 画得出就是平面图
```

---

## 六、平面图的对偶图

平面图 G 的每个面变成一个顶点，跨面的边变成对偶图的边。

- 对偶图中，**割集 ⇔ 回路**（对偶关系）
- 面数 r → 对偶图顶点数
- 对偶图的对偶图 = 原图

---

## 七、总结表

| 定理 | 类型 | 用法 |
|---|---|---|
| 欧拉公式 n−m+r=2 | 恒等式 | 数面数、边数、顶点数 |
| m ≤ 3n−6 | 必要条件 | 秒杀过密图 |
| m ≤ 2n−4 | 必要条件 | 秒杀二分图 |
| Kuratowski 定理 | **充要条件** | 找 K₅/K₃,₃ 子图 = 定案 |

---

**一句话：平面图 = 能无交叉地画出来。判平面图三步走——先算边数（秒否），再找 K₅/K₃,₃ 同胚（定案），画得出就是。**
