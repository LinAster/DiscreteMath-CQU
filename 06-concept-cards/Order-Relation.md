# 序关系（Order Relation）

---

## 一、什么是序关系

> 集合上的一个二元关系，满足**自反 + 反对称 + 传递**，就叫**偏序关系**（partial order），记作 ⪯。

| 性质 | 含义 | 例子（≤） |
|---|---|---|
| 自反 | x ⪯ x | 3 ≤ 3 ✓ |
| 反对称 | x ⪯ y 且 y ⪯ x → x = y | 3≤5 且 5≤3 → 3=5（不可能） |
| 传递 | x ⪯ y 且 y ⪯ z → x ⪯ z | 2≤5 且 5≤7 → 2≤7 ✓ |

满足这三条的二元关系，配上一个集合，就叫**偏序集** (A, ⪯)。

---

## 二、三个层次的序关系

### 2.1 偏序（Partial Order）

部分元素之间可以比大小，部分不能。

**例子：整除关系 | 在 {2, 3, 4, 6, 12} 上**

```
2|4 ✓, 4|12 ✓, 2|12 ✓
2|3 ✗, 3|4 ✗    ← 有些对不可比
```

> 偏序 = 不要求"任意两个都能比"。

### 2.2 全序（Total Order / Linear Order）

偏序 + **任意两个元素都可比**。

**例子：≤ 在 ℤ 上** —— 任意两个整数，不是 a≤b 就是 b≤a。

**反例：整除** —— 2 和 3 不可比，不是全序。

> 全序 = 偏序 + "任意两个都能排出先后"。

### 2.3 良序（Well Order）

全序 + **每个非空子集都有最小元**。

**例子：(ℕ, ≤)** —— 任何非空子集都有最小自然数 ✓

**反例：(ℤ, ≤)** —— 负整数子集 {..., −3, −2, −1} 没有最小元 ✗

> 良序 = 全序 + "不能无限往下走"。

### 层次关系

```
良序 ⊂ 全序 ⊂ 偏序
(最强)         (最弱)
```

## 三、偏序集的关键元素

以整除关系在集合 {2, 3, 4, 6, 12} 上为例：

| 概念 | 定义 | 本例 |
|---|---|---|
| 极大元 | 没有元素比它大 | 12 |
| 极小元 | 没有元素比它小 | 2, 3 |
| 最大元 | 比所有元素都大 | **无**（没有元素同时整除 2 和 3） |
| 最小元 | 比所有元素都小 | **无** |
| 上界 | 比某个子集所有元素都大 | {2,3} 的上界：6, 12 |
| 下界 | 比某个子集所有元素都小 | {4,6} 的下界：2 |
| 上确界 | 最小上界 | {2,3} 的上确界：**6** |
| 下确界 | 最大下界 | {4,6} 的下确界：**2** |

### 极大元 ≠ 最大元

- **极大元**：没人比我大（可以有好几个）
- **最大元**：我比所有人都大（最多一个）

上例：极大元有 12（只有它），但最大元不存在——因为 12 不整除任何东西？不对，12 本身比其他都大，它就是最大元...

等等——这是整除关系，a ⪯ b 意思是 a|b。所以：
- 12 能被 2,3,4,6 整除但不能整除它们，12 是极大元
- 但 12 不是最大元，因为需要 12 能整除 2？不对——最大元要求"对任意 x, x ⪯ 最大元"，即任意 x 整除最大元。2|12 ✓, 3|12 ✓, 4|12 ✓, 6|12 ✓, 12|12 ✓ → **12 就是最大元**！

重新整理：

| 元素 | 分析 |
|---|---|
| 极大元 | 12 |
| 极小元 | 2, 3（没有比它们更小的能整除它们） |
| 最大元 | 12（所有元素都整除 12） |
| 最小元 | **无**（2 不能整除 3，3 不能整除 2） |

## 四、哈斯图（Hasse Diagram）

偏序集的"简化关系图"——去掉自环和传递边。

### 画法规则

1. 小在下，大在上
2. 去掉自环（自反不画）
3. 去掉传递边（由"链条"自动表达）
4. 只画**盖住**关系（直接覆盖，中间没有其他元素）

### 整除关系 {(2,3,4,6,12), |} 的哈斯图

```
        12
       /  \
      4    6
       \  /
        2
         \
          3
```

- 2 盖住... 没有谁被 2 直接盖住？实际上 2→4, 2→6（都是直接整除）
- 3→6
- 4→12, 6→12

画成：

```
        12
       /  \
      4    6
       \  / \
        2   3
```

- 2 和 3 之间没有线 → 不可比 ✓
- 没有自环 ✓
- 没有跳级边（2→12 不画，传递可得）✓

## 五、常见序关系举例

| 序关系 | 集合 | 类型 |
|---|---|---|
| ≤ | ℕ, ℤ, ℝ | 全序（ℕ 上还是良序） |
| 整除 \| | ℕ⁺ | 偏序，不是全序 |
| ⊆ | P(A) | 偏序 |
| 字典序 | 字符串 | 全序 |

---

**一句话：偏序 = 自反 + 反对称 + 传递；全序 = 偏序 + 任意两个可比；良序 = 全序 + 非空子集必有最小元。哈斯图画偏序集，小在下大在上，去自环去传递。**
